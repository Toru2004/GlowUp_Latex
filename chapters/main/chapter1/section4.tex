\section{Công cụ sử dụng}
Các công cụ chính được sử dụng trong đề tài Xây dựng website bán mỹ phẩm kết hợp AI hỗ trợ tư vấn da mặt trị mụnvà chatbot gợi ý sản phẩm
	
	\textbf{IDE và lập trình:} Visual Studio Code.
	
	\textbf{Frontend:} Html, CSS, NuxtJS, Tailwind và jwt-decode.
	
	\textbf{Backend:} Node.js, ExpressJS, bcrypt, jsonwebtoken, dotenv, cors và multer.
	
	\textbf{Cơ sở dữ liệu:} SQL Server.
	
	\textbf{Quản lý và triển khai:} Git và GitHub.
	
	\textbf{Debug và chạy hệ thống:} Chrome, Edge, npm run dev và npm run build.
