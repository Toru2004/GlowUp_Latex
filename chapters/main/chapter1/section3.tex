\section{Công cụ sử dụng}
Các công cụ chính được sử dụng trong đề tài Xây dựng website bán mỹ phẩm kết hợp AI hỗ trợ tư vấn da mặt trị mụn và chatbot gợi ý sản phẩm
	
	\textbf{IDE và lập trình:} Visual Studio Code.
	
	\textbf{Frontend:} Nuxt.js.
	
	\textbf{Backend:} ExpressJS.
	
	\textbf{Cơ sở dữ liệu:} SQL Server.
	
	\textbf{Quản lý và triển khai:} GitHub, Postman.
	
	\textbf{Debug và chạy hệ thống:} Chrome, Edge, npm start và npm run build.
